squid介绍

​ squid是支持HTTP，HTTPS，FTP等Web的缓存代理。它通过缓存和重复使用频繁请求的网页来降低带宽并缩短响应时间。Squid拥有广泛的访问控制，并且是一款出色的服务器加速器 它可以在大多数可用的操作系统上运行，包括Windows，并在GNU GPL下获得许可。
